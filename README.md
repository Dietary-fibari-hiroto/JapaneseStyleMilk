# 🗣️ langbate — 英語ディベートで、話す力を鍛える

> **ディベートで日常会話レベルの壁を超えよう**

---

## 🔍 Overview

**langbate** は、英語スピーキング力を鍛えたい日本人大学生のためのディベート特化型Webシステムです。  
「話す力」と「論理的思考力」を伸ばすことで、**外資系企業での活躍**や**海外留学**を目指す人たちを支援します。

- 🎯 **即興英語ディベート × AI評価**
- ⏱️ **約20分で完結する3ターン制（意見→反論→結論）**
- 📈 **構成・語彙・表現を自動フィードバック**
- 🎲 **お題と相手は毎回ランダム！学習の幅を広げる設計**

---

## 🧠 Concept

> **「英語を話す体験」そのものを、もっと習慣的で気軽に。**

langbateは、**評価されることで伸びを実感できる学習体験**を提供します。  
単なるチャットや通話ではなく、「構成・論理」を意識した会話によって、**使える英語力**を定着させます。
